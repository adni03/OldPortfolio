Projects: #

Credit Card Risk Analyzer: #

  • Conducted Exploratory Data Analysis (EDA) on the Credit Card Approval dataset (UCI ML Repository)
  • Cleaned data by imputing NaN values with the mean of the dataset, and encoded non-numeric data using LabelEncoder
  • Scaled the data using MinMaxScaling and fit the data to a Logistic Regression model
  • Performed Hyperparameter Tuning using GridSearchCV to obtain an accuracy of 86.4%

Facial Emotion Recognition: #

  • Developed a Convolutional Neural Network (CNN) to predict human emotions

TicTacToe AI: #

  • Constructed a Machine Learning model to learn to play NxN TicTacToe
  • Extracted features from the current state of the board and used the Least Mean Squared weight update rule
